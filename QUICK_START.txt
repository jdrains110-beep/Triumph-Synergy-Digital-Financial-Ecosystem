## 🆘 TROUBLESHOOTING

**Blank screen?**
- Clear browser cache
- Check Vercel deployment status
- Open browser console (F12) for errors

**Payment fails?**
- Verify PI_API_KEY is set in Vercel
- Check it's the FULL key (not truncated)
- Ensure it matches mainnet/testnet selection

**Can't access domain?**
- Wait for Vercel deployment to complete
- Try different browser
- Clear DNS cache

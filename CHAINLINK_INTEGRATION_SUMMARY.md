## 📞 API DOCUMENTATION

### Complete Chainlink API Reference

**Base URL**: `https://triumph-synergy.vercel.app/api/chainlink`

#### Get Price Feed
```bash
curl "https://api.example.com/api/chainlink/prices?pair=PI/USD&action=price"
```

#### Get Multiple Prices
```bash
curl "https://api.example.com/api/chainlink/prices?pairs=PI/USD,XLM/USD&action=prices"
```

#### Request VRF
```bash
curl -X POST "https://api.example.com/api/chainlink/vrf" \
  -H "Content-Type: application/json" \
  -d '{"action":"request_vrf"}'
```

#### Get VRF Result
```bash
curl -X POST "https://api.example.com/api/chainlink/vrf" \
  -H "Content-Type: application/json" \
  -d '{"action":"get_vrf_result","requestId":"vrf_request_xyz"}'
```

#### Check Health
```bash
curl "https://api.example.com/api/chainlink/prices?action=health"
```
